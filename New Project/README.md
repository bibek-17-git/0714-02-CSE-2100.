# Smart Backup Utility

Course Code: 0714 02 CSE 2100 | Advanced Programming Laboratory

## Project Structure

```
snake-game/
├── src/
│   ├── main/
│   │   └── main.c          # Entry point only
│   ├── core/
│   │   ├── settings.c      # Load/save config
│   │   ├── fileops.c       # File/folder list & dialogs
│   │   └── backup.c        # Backup engine & timer
│   ├── rendering/
│   │   └── rendering.c     # GTK window, settings dialog, log viewer
│   └── utils/
│       └── utils.c         # Status bar & notification helpers
├── include/
│   ├── settings.h
│   ├── ui.h
│   ├── backup.h
│   ├── fileops.h
│   └── utils.h
├── docs/
├── assets/
├── build/                  # Generated by make (gitignored)
├── Makefile
└── README.md
```

## Build (MSYS2 / MinGW-w64)

```bash
make
```

## Clean

```bash
make clean
```

## Dependencies

- GTK+ 3.0 (`pkg-config gtk+-3.0`)
